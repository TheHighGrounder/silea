## Install

This is a package internally used by [Silea CLI](https://docs.sileajs.com/silea-cli)

You'll find it as a dependency of your Silea project after you create it by doing

```sh
npx silea create [project-name]
```

> [More info on `npx silea create`](https://docs.sileajs.com/silea-cli/create)
